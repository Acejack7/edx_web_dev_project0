Below you can find modules that I recently used and a few words about them:
os
Any operations on files and folders - removing, copying, renaming; checking if the element is a certain file, changing working directory;
re
Designed for using regular expressions in python;
xlsxwriter/openpyxl
xlsxwriter is used to create excel files using python; excel files cannot be opened by xlsxwriter but xlsxwriter has more functionalities regarding formatting then openpyxl; on the other hand openpyxl can open excel files and edit them but it cannot, for example, create multicolor font in one cell;
zenpy
For zendesk ticketing platform; Zendesk has great API and with this module I could extract a lot of various data about tickets and insert them in excel file;
xml.dom
I specifically used minidom from this module and it was enough to parse XML files and perform changes on them;
